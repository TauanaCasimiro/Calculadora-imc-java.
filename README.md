# Calculadora de IMC em Java

Este é um projeto simples de lógica pura que calcula o Índice de Massa Corporal (IMC).

## Como funciona

O programa solicita ao usuário:
1. O peso (ex: 70,5)
2. A altura (ex: 1,75)

Após os dados, o sistema realiza o cálculo matemático e exibe a classificação (Abaixo do peso, Peso normal ou Acima do peso).

## Tecnologias Utilizadas

- Java
- Scanner (para entrada de dados)
- Estruturas condicionais (if/else)
